# 素材需求列表 (Asset Requirements List)

为了开发 Web 版杀戮尖塔的 MVP (最小可行性产品)，我们需要以下素材。
请按照建议的**路径**存放文件，或提供文件给我，由我来放置。

> **提示**: 尺寸建议仅供参考，只要比例正确，稍大或稍小均可适配。格式首选 PNG (带透明通道)。

## 1. 界面与通用 UI (UI Assets)
**目标路径**: `public/assets/ui/`

| 素材名称 | 描述 | 建议尺寸 (px) | 格式 | 建议文件名 |
| :--- | :--- | :--- | :--- | :--- |
| **能量图标** | 显示左上角的能量球 | 128x128 | PNG (透明) | `energy_orb.png` |
| **生命值图标** | 玩家/敌人血条旁的心形图标 | 64x64 | PNG (透明) | `heart_icon.png` |
| **格挡图标** | 盾牌形状，用于显示护甲 | 64x64 | PNG (透明) | `block_icon.png` |
| **金币图标** | 顶部栏的金币符号 | 64x64 | PNG (透明) | `gold_icon.png` |
| **结束回合按钮** | 大号按钮，处于激活/禁用状态 | 256x100 | PNG | `end_turn_btn.png` |
| **意图图标-攻击** | 敌人头顶显示的攻击意图 (剑) | 64x64 | PNG (透明) | `intent_attack.png` |
| **意图图标-防御** | 敌人头顶显示的防御意图 (盾) | 64x64 | PNG (透明) | `intent_defend.png` |
| **意图图标-Buff/Debuff** | 敌人施加状态的意图 (魔法光晕) | 64x64 | PNG (透明) | `intent_buff.png` |
